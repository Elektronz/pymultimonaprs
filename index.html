PyMultimonAPRS

HF 2 APRS-IS Gateway

Download this project as a .zip file Download this project as a tar.gz file

This tool receives APRS packages using a soundcard or the RTL-SDR and redirects them to the APRS-IS network (which is the data-source for aprs.fi).

It is also capable of sending beacon packages to the APRS-IS network:

  • Position beacons
  • Dynamic/Static status beacons
  • Dynamic weather beacons
